Sales

Sales Orders can be managed by the shop owner from the back-office application (Zed). In this document we’ll describe the functionality offered by the Sales module that will help you manage the orders submitted in your shop.

The Order Management System strongly correlates with the state machines you have defined in your system. This user manual only describes the capabilities of the Sales module, without referring to specific use case scenarios.

1. View Order List

On the left side navigation bar, select Sales > Orders.

After clicking on the orders menu item, you’ll be able to see a table containing the orders submitted in your shop. You can sort the list of orders by the timestamp when they were submitted, in ascending or descending order. Also, you can select how many entries should be listed on each page.

The Orders List screen enables you to perform search : the string you enter in the box will be searched among the fields displayed in the table.

2. View Order Details

By clicking on the corresponding View button for a sales order, you can see the details and manage the order.

The first section of the order details screen contains the Order Overview; here you’ll see the information regarding the current order which is described in more details in the sections below.

On this section, the back-office user is able to add a comment; the message can be seen by the other back-office users that have access rights to the Sales Order menu item.

The next section contains the matching states that can be triggered, either for the entire order of for one or more items contained. A matching state means that there is a transition between it and the current state of the order that can be manually triggered.

The matching states can be triggered by order item; in the Trigger event column you can see the buttons that correspond to each matching state. By clicking the button of a matching state, the state machine will move to that state for the associated order item.

Example: you can trigger a refund for one item

By clicking the Show more link in the State history column, you can see the entire history for each item included in the order (the states in which the item has been, together with the timestamp when the transition to that state took place).

On the Totals section you can see the calculation details of the grand total; this calculation included the contained items together with their gross price, the shipment expense and the discounts that are applied.

On the Shipment section you are able to see the carrier and shipment method selected by the customer for the current order, together with the price.

On the Discount section you can see listed the discounts that are applied to the current order : on the left side are listed the vouchers and on the right side the cart rules. For more details about discounts you can consult the Discounts User Manual.

On the Addresses section you can see the details of the billing and shipping address for the sales order. By clicking the Edit button on either of the addresses you are able to edit the details of the address.

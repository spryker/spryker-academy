In Spryker, there are two types of discounts that can be defined :

  • Cart Rules
    This discount is contained in the cart and doesn’t need any action to be performed by the customer. This discount is automatically applied if the decision rules attached to it are being satisfied and if the discount is currently active; also, this discount can be redeemed only during its validity time interval.
  • Vouchers
    To redeem this type of discount, the customer must enter a voucher code. Similar to cart rules discount, the discount is applied only if the linked decision rules are satisfied and if it’s active and valid.

Regarding the way a discount is calculated, there are 2 types of discounts :

  • percentage discount
  • fixed amount discount

Creating a Voucher

There are two steps to creating a voucher:

  1. Create the voucher by setting up the rules on which this discount can be redeemed
  2. Generate the voucher codes that can be used by the customers to redeem this discount

Create the Voucher

Vouchers
A voucher is defined by a validity interval, it can be set active/inactive and can have 0 or many decision rules attached. For the voucher to be redeemable, you need to generates voucher codes for it.

To create a voucher for your shop:

  1. On the left side navigation bar, select Discount > Voucher Codes

  1. Click the Create Voucher button on the top right corner

Next, configure when the voucher will be redeemable and which conditions should be satisfied.

  1. Enter a name, a description and select a pool category for the voucher

The name and the description should be meaningful for the ones using the store back-office application.

The pool categories are used for grouping/labelling discounts. E.g.: the Customer Care pool category should contain vouchers that have the purpose to be used by the customer care department. The purpose of the pool categories is to better organize the vouchers you are creating.

The collector plugin specifies the level on which the voucher will be applied; you can choose from : item, order expense, item expense, item product option and aggregate (all the items). The associated values stands for the minimum value so that the discount can be applied

Collector Plugin

A voucher can have associated multiple collector plugins. The plugins can be combined using the AND or OR logic operators.

E.g. : the voucher can be redeemed if the order expense value is equal or higher than 25€.

Edit Voucher Four

7. Add the set of decision rules for your new voucher

A voucher can have one or more decision rules linked to it. The voucher can be redeemed if and only if every decision rule linked to it is satisfied.

Decision Rule

A decision rule, as it name says, decides if the discount can be redeemed. A value can be passed as an argument to the decision rule.

If you don’t need to add a decision rule on your voucher, you must select the Voucher option from the list on this step. No value needs to be provided for this option.

8. Set the active/inactive property for your voucher

Even if a voucher is valid and the decision rules are satisfied, a voucher can be redeemed only if it’s currently active. The voucher can be activated after creation as well, by updating this property.

9. Set up if the voucher can be combinable

Combinable means that the discounts from multiple vouchers or cart rules or a combination of both may be applied. If this is the case, the combination resulting in the highest discount is applied to the customers order. Furthermore, this means that the vocher/cart rule that is marked as combinable will always be applied. If a non-combinable voucher/cart rule offers a higher discount than the one marked as “combinable”, then the combination of the combinable and the non-combinable (with the higher discount) will be applied.

This is everything you need to do for creating the voucher. Hit the Save button so that we can move to the next step to create the voucher codes.

Generate the Voucher Codes

1. Click View Vouchers button in the top right corner to see the list of vouchers you have

Here you have the option to edit existing vouchers, view voucher codes (if they were already created) or generate new voucher codes. Being that we just created the voucher, the list will be empty at this moment.

You can generate a single voucher code or multiple voucher codes.

New Voucher

If you choose to generate multiple voucher codes, on the next step you’ll be prompted to enter the number of codes you want to generate.

Voucher Code Quantity

2. Enter a Custom Code

The Custom Code is a string that will be contained in the voucher code(e.g. christmas16).

In the case that you choose to only create one voucher code, the code will be as defined by you. If you choose to create multiple voucher codes at once, you need to add a random generated code length to the custom code (next step).

Voucher Custom Code

3. Set the Random Generated Code Length

When generating multiple voucher codes, a random number of characters will be added to the entered custom code. This number has to be set according to the number of codes you need to generate, so that it is possible that distinct voucher codes get generated.

Voucher Code Random

4. Set the Maximum number of uses

For the new voucher codes a maximum number of uses needs to be defined.

Creating a Cart Rule

A cart rule is a discount that is contained in the cart and needs no input from the customer. If the discount is active, valid and the cart satisfies the linked decisions rule, then it’s automatically calculated for the order.

Creating a Cart Rule

The steps for creating a cart rule are similar to the ones for creating a voucher, except you don't need to generate voucher codes for it.

You can follow the steps described below to create a new cart rule discount.

1. From the left side navigation menu select Discount > Cart Rules

2. Click on the Add Cart Rule button in the top right corner
